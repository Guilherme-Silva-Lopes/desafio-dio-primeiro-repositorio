O que é um fluxograma?
R: Fluxograma é uma ferramenta utilizada para representar graficamente o algoritmo, isto é, a sequência lógica e coerente do fluxo de dados.

 O que são variáveis?
R: Na progamação, uma variável é um objeto (uma posição, frequentemente localizada na memória) capaz de reter e representar um valor ou expressão
/ou 
Uma variável é um espaço na memória do computador destinado a um dado que é alterado durante a execução do algoritmo.

Tipos de variáveis
R: As variáveis e as constantes podem ser classificadas basicamente de quatro tipos: Numéricas, Caracteres, Alfanuméricas ou Lógicas.

O que é uma constante?
R: As constantes são valores imutáveis e não são alterados durante a vida útil do progama.

AULA 4 - TOMADAS DE DECISÕES E EXPRESSÕES

Expressões aritméticas
R: São expressões que utilizam operadores aritméticos e funções aritiméticas envolvendo constantes e variáveis.

Exemplo:
50+50
Total+50

Operadores sritiméticos
Soma: +
Subtração: -
Multiplicação: *
Divisão: /
Potenciação: ^
Porcentagem: %

Expressões literais
R: São expressões com constantes e/ou variáveis que tem como resultado valores literais. Iremos utilizar as expressões literais na atribuição de valor para uma varável ou constante.

Exemplos:
nome= "José da Silva"
nome <-- "José da Silva"
media= (nota1+nota2+nota3+nota4)/4

Operadores relacionais (Expressões)
R: São expressões compostas por outras expressões ou variáveis numéricas com operadores relacionais.
As expressões relacionais retornam valores lógicos (verdadeiro/falso).

Operadores relacionais
Maior que: >          Exemplo: x > y
Maior ou igual: >=    Exemplo: x >= y
Menor que: <           etc...
Menor ou igual:<=
igualdade: ==
diferente de: != ou <>     Exemplo x != y ou x <> y

Tomadas de decisão
R: Quando escrevemos progamas, geralmente ocorre a necessidade de decidir o que fazer dependendo de alguma condição encontrada durante a execução.

AULA 7 - O QUE SÃO LINGUAGENS DE PROGAMAÇÃO?

Linguagem de progamação
R: Linguagem de progamação é uma linguagem escrrita e formal que especifica um conjunto de instruções e regras usadas para gerar progamas (software). Um software pode ser desenvolvido para rodar em um computador, dispositivo móvel ou em qualquer equipamento que permita sua execução.

"A função das linguagens de progamação é servir como um meio de comunicação ebtre computadores e humanos"


DIVISÇOES DAS LINGUAGENS

Linguagens de alto nível:: São aquelas cuja sintaxe é mais parecida com a da linguagem humana.

Linguagens de baixo nivel: São aquelas cuja sintaxe é mais parecida com a liguagem de máquina. elas requerem que você tenha conhecimento sobre arquitetura de hardware para ser usada.

Linguagens compiladas: São linguagens de progamação em que o código fonte, é executado diretamente pelo sistema operacional ou pelo processador, após ser traduzido por meio de um processo chamado compilação.

Linguagens interpretadas: São linguagens de progamação em que o código fonte é executado por um progama de computador chamado interpretador, que em seguida é executado pelo sistema operacional ou processador.

O que é o Portugol?
R: Portugol é uma pseudolinguagem que permite ao leitor desenvolver algoritmos estruturados em português de forma simples e intuitiva, independentemente de linguagem de progamação.

APRENDA A USAR DESVIOS CONDICIONAIS E BOAS PRATICAS EM PORTUGOL

Desvio condicional - se
R: É utilizada a palavra reservada "se", a condição a ser testada entre parenteses e as instruções que devem ser executadas entre chaves caso o desvio seja "verdadeiro".

se-senao
R: Agora vamos imaginar que se a condição for "falsa" um outro conjunto de comandos deve ser executado.
Quando iremos encontrar esta situação?

Exemplo:

se (media>=7) {
    escreva("Parabéns!! Você passou!!")

}

senao {
   escreva("infelizmente você foi reprovado")  

}

caso
R: Este comando é similar aos comandos "se" e "senao", e reduz a complexidade na escolha de diversas opções.
Apesar de suas similaridades com o "se", ele possui algumas diferenças. Neste comando não é possivel o uso de operadores lógicos, ele apenas trabalha com valores definidos.

TRABALHANDO COM LAÇOS DE REPETIÇÃO EM PORTUGOL

Laços de repetição
R: Dentro da lógica de progamação é uma estrutura que permite executar mais de uma vez o mesmo comando ou conjunto de comandos, de acordo com uma condição ou com um contador.

Exemplo

funcao inicio()
{
       inteiro contador, limite, resultado
       contador = 0
       resultado = 10
       faca
        {
            resultado = 9*contador
            escreva("9x" + contador + "=" + resultado + "\n" )
            contador ++    // significa contador + 1
        } enquanto(contador<=limite)
}
